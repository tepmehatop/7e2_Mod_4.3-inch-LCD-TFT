## 📝 ЧТО ВАМ НУЖНО СДЕЛАТЬ:

### Шаг 1: Компиляция
1. Открыть Arduino IDE
2. Открыть файл `7e2_Mod_LCD4.3_v1.3.ino`
3. Нажать **Verify** (галочка) или Ctrl+R
4. Убедиться что компиляция прошла **БЕЗ ОШИБОК**

**Ожидаемый результат:**
```
Sketch uses XXXXX bytes (XX%) of program storage space.
Global variables use XXXX bytes (XX%) of dynamic memory.
```

**Если ошибки:**
- Проверьте что файл `SerialEmulation.ino` в той же папке
- Проверьте что все файлы проекта на месте

---

### Шаг 2: Загрузка на Arduino
1. Подключить Arduino Mega 2560 по USB
2. Выбрать правильный порт: **Tools → Port → [ваш порт]**
3. Выбрать плату: **Tools → Board → Arduino Mega or Mega 2560**
4. Нажать **Upload** (стрелка вправо) или Ctrl+U
5. Дождаться окончания загрузки

**Ожидаемый результат:**
```
avrdude: XXXX bytes of flash written
avrdude done.  Thank you.
```

---

### Шаг 3: Открыть Serial Monitor
1. Нажать **Tools → Serial Monitor** или Ctrl+Shift+M
2. В правом нижнем углу выбрать:
   - Скорость: **115200 baud**
   - Окончание строки: **Newline** или **Both NL & CR**

**Ожидаемый результат:**
Вы должны увидеть приветственное сообщение и справку по командам:
```
===========================================
ELS v1.3 - Serial Emulation Mode
===========================================
Доступные команды:

=== ДЖОЙСТИК ===
JL  - Джойстик Влево
JR  - Джойстик Вправо
...
```

### При правильной работе:
- ✅ Каждая команда выводит подтверждение

- ✅ Параметры меняются (кнопки Up/Down/Left/Right)

- ⚠️ LCD не подключен - экран не будет показывать изменения (мы еще не подключили)
- ⚠️ Моторы не крутятся - драйверы не подключены (подключим позже)
- ⚠️ Энкодер шпинделя не читается - не подключен (подключим позже)
- ⚠️ Звуковые сигналы могут не работать - зуммер не подключен

## ❌ TROUBLESHOOTING (Если что-то не работает):

### Проблема: Компиляция завершилась с ошибкой
**Решение:**
- Убедитесь что файл `SerialEmulation.ino` в той же папке что и основной файл
- Проверьте что все файлы проекта (.ino) на месте
- Попробуйте перезапустить Arduino IDE

### Проблема: Serial Monitor показывает "мусор"
**Решение:**
- Установите скорость **115200 baud** в правом нижнем углу Serial Monitor
- Перезагрузите Arduino (кнопка Reset на ��лате)

### Проблема: Команды не реагируют
**Решение:**
- Проверьте окончание строки: должно быть **Newline** или **Both NL & CR**
- Команды должны быть в ВЕРХНЕМ РЕГИСТРЕ (JL, BR, M1, не jl, br, m1)
- После ввода команды нажимайте Enter

### Проблема: INFO не показывает изменений
**Решение:**
- Убедитесь что команда выполнилась (есть подтверждение "Эмуляция: ...")
- Некоторые изменения видны только в определенных режимах (например резьба только в Mode Thread)
