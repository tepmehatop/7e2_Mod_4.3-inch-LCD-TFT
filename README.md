# ELS Display — Замена монитора электронного суппорта токарного станка

Проект представляет собой замену штатного текстового дисплея LCD2004 на современный сенсорный 4.3" цветной дисплей для системы **ELS (Electronic Lathe System / Электронный суппорт токарного станка)**.

---

## Авторство и источники

Оригинальная прошивка Arduino для ELS разработана **Олегом А.**  
Исходный код взят : **[https://www.chipmaker.ru/topic/268878/](https://www.chipmaker.ru/topic/268878/)**



Данный проект — модификация оригинальной прошивки с заменой дисплея:
новый ESP32-S3 + 4.3" сенсорный дисплей вместо LCD2004.

## Отображение данных на новом дисплее

- **4.3" цветной сенсорный дисплей** вместо текстового LCD2004
- Все 7 режимов работы: M1 Подача / M2 аПодача / M3 Резьба / M4-M5 Конус / M6 Шар / M7 Делитель
- Отображение позиций Z и X в реальном времени
- Управление режимами, подрежимами и параметрами через тачскрин
- Встроенный джойстик на тачскрине для ручного перемещения осей
- Индикаторы концевых выключателей (Z: ←→, X: ↑↓)
- 67-позиционная таблица резьб: метрические / дюймовые / G-трубные / K-трубные
- RPM-лимит для нарезки резьбы в зависимости от шага

## Подключение Arduino Mega ↔ JC4827W543


| Arduino Mega       | JC4827W543 (P1) | Сигнал                      |
| ------------------ | --------------- | --------------------------- |
| Pin 51 (SS_TX)     | GPIO44 (RX)     | UART данные → дисплей       |
| Pin 52 (SS_RX)     | GPIO43 (TX)     | UART данные → Arduino       |
| GND                | GND             | Земля                       |
| 5V / 3.3V          | —               | Питание от своего источника |

> Скорость: **57600 baud** (SoftwareSerial), протокол: `<CMD:VALUE>\n`
> Пины 18/19 возвращены ручному энкодеру. Пины 20/21 полностью освобождены под энкодер шпинделя.

### Подключение старого LCD2004 (временно, для параллельного тестирования)

| Arduino Mega | LCD2004 I2C | Сигнал       |
| ------------ | ----------- | ------------ |
| Pin 11       | SDA         | I2C данные   |
| Pin 12       | SCL         | I2C тактовый |
| GND          | GND         | Земля        |
| 5V           | VCC         | Питание      |

> Bit-bang I2C (программный). Адрес: 0x27 (или 0x3F).

## Сборка и прошивка

### Arduino Mega

```bash
# Через Arduino IDE или arduino-cli:
arduino-cli upload --fqbn arduino:avr:mega --port /dev/cu.usbserial-XXXX /path/to/project
```

### ESP32-S3 (JC4827W543)

```bash
cd ELS_Display
~/.platformio/penv/bin/pio run -e jc4827w543 --target upload
```
